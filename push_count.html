push/count.json 概要

通知センターにある未読の通知件数を取得します。

リクエスト

パラメータ 値 説明

レスポンス

フィールド名 説明
link フレンドリクエストの通知未読件数
message メッセージの通知未読件数
other その他自分宛の通知未読件数

サンプルリクエスト

$ curl http://example.com/sns/api.php/push/count.json?apiKey=5ad9d5a6a11c5f13ca5901a7c6e1d89ea991dc195666de393a61c808c61d19a5
