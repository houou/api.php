diary/delete.json 概要

日記削除

リクエスト

パラメータ 値 説明
id *必須 integer 削除したい日記記事のIDを指定します。

レスポンス

フィールド名 説明
id 削除した記事ID

サンプルリクエスト

$ curl http://example.com/api.php/activity/delete.json?id=20&apiKey=5ad9d5a6a11c5f13ca5901a7c6e1d89ea991dc195666de393a61c808c61d19a5

サンプルレスポンス

{
    "status": "success",
    "data":{
        "id":20,
        },
    }
}
