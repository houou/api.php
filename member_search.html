member/search.json 概要

SNSメンバー情報を検索して取得してきます。

リクエスト

パラメータ 値 説明
target *必須 string 取得したいメンバー情報の、ターゲットを指定します。
memberと指定するとメンバーそのものの情報が、
friendと指定するとメンバーのフレンドリストが、
communityと指定するとコミュニティの参加者リストが取得できます。
target_id *必須 integer 取得したいメンバー情報の、ターゲットとなるID(コミュニティID、メンバーID)を指定します。

レスポンス

フィールド名 説明
id メンバーIDを返します。
profile_image メンバーのプロフィール画像を絶対URLで返します。プロフィール画像が登録されていない場合は、no_image.gifを絶対URLで返します。
name メンバーのニックネームを返します。
friend 自分とフレンドリンクであればtrue、そうでなければfalseを返します。
blocking 自分がアクセスブロックしているメンバーであればtrue、そうでなければfalseを返します。
self このメンバー情報は自分自身のものであればture、そうでなければfalseを返します。
friends_count メンバーのフレンドリンクの数を返します
self_introduction メンバーのプロフィールにある自己紹介を返します。プロフィール項目に自己紹介が設定されていない場合はfalseを返します。

サンプルリクエスト

$ curl http://example.com/sns/member/search.json?target=friend&target_id=1234&apiKey=5ad9d5a6a11c5f13ca5901a7c6e1d89ea991dc195666de393a61c808c61d19a5
$ curl http://example.com/sns/member/search.json?target=community&target_id=123&apiKey=5ad9d5a6a11c5f13ca5901a7c6e1d89ea991dc195666de393a61c808c61d19a5
