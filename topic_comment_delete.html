topic_comment/delete.json 概要

トピックコメント削除

リクエスト

パラメータ 値 説明
id *必須 integer コメントのIDを指定します。

レスポンス

フィールド名 説明
id コメントID
member 投稿したメンバーの情報(詳しくは レスポンスの共通仕様 を参照)
body 投稿したトピックの本文
created_at 投稿時間
deletable コメントを削除可能かどうかのフラグ

サンプルリクエスト

$ curl http://example.com/api.php/topic_comment/delete.json?apiKey=cdc4042edf74db746f5bf550039bf50979da2566aa514fc0aac8f57c56398be5&id=3
