topic/delete.json 概要

トピック削除

リクエスト

パラメータ 値 説明
id *必須 integer トピックIDを指定します。

レスポンス

フィールド名 説明
id トピックID
community_id コミュニティID
community_name コミュニティの名前
member トピック所有者メンバーの情報(詳しくは レスポンスの共通仕様 を参照)
name トピックのタイトル
body トピックの本文
created_at トピックの作成時間
images トピックに添付された画像のリスト
editable トピックが編集可能かどうかのフラグ

サンプルリクエスト

$ curl http://example.com/api.php/topic/delete.json?apiKey=cdc4042edf74db746f5bf550039bf50979da2566aa514fc0aac8f57c56398be5&id=1
