message/list.json 概要

メッセージの一覧を取得します。

リクエスト

パラメータ 値 説明
type string 取得したいメッセージのタイプを指定します。receivesenddraftdustが指定可能です。
max_id integer 取得したいメッセージIDの最大値を指定します。
since_id integer 取得したいメッセージIDの最小値を指定します。
count integer 取得したいメッセージの、取得件数を指定します。デフォルトは20。
detail boolean メッセージの詳細まで出力する場合はtrueを、そうでない場合はfalseを指定します。

レスポンス

フィールド名 説明
id メッセージのID（連番）
member メンバーの情報（詳しくは レスポンスの共通仕様 を参照）
title メッセージの件名
body メッセージの本文(リクエスト時にdetail=trueとしていた場合のみ有効)
created_at 作成された時間(送受信時間)
