member/friend_request.json 概要

フレンドリクエストを申請します。

リクエスト

パラメータ 値 説明
member_id *必須 integer フレンドリクエストを承認、あるいは拒否する相手のメンバーIDを指定します。
reject boolean フレンドリクエストを拒否したい場合は、このリクエストの値をtrueに指定します。

レスポンス

フィールド名 説明
message statusフィールドがerrorのときにメッセージを返します。

サンプルリクエスト

$ curl http://example.com/sns/member/friend_accept.json?member_id=1234&reject=true&apiKey=5ad9d5a6a11c5f13ca5901a7c6e1d89ea991dc195666de393a61c808c61d19a5
